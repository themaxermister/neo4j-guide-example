# Using neo4j

## 1. Selecting query

### a) Get all nodes

```sql
MATCH (n) RETURN n
```

### b) Get nodes by entity type

``` sql
MATCH (n:<TYPE>) RETURN n
```

### c) Get property 

``` sql
MATCH (n:<TYPE>) RETURN n.<PROPERTY> AS property_name, n.<PROPERTY> as property_name
```

### d) Get nodes by condition

```sql
MATCH (n:<TYPE>) WHERE <CONDITION> RETURN n
```
